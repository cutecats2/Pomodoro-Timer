# Cat Pomodoro Timer
A minimalist desktop application applying the pomodoro technique, which is a time-management method 
which alternated between work and break sessions. The application gives visual feedback based on what 
the user is doing and also provides encouraging messages to the user. This application is build with React, TypeScript,
and Electron. It also utilizes CSS to apply custom styling and layout. 
## Goals: 
- Combine React with Electron
- Learn desktop app packaging
